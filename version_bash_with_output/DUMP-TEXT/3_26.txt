    過去 70 年間 に 300 人 以上 の カトリック 教会 の 「 肉食 聖職 者 」 たち が 、 1 , 000 人 以上 の 子ども たち を 虐待 し た 、 と 厳しく 批判 し た 報告 書 を 州 司法 長官 が 
    発表 し た 。 

    1 , 400 ページ に およぶ その 報告 書 は 、 ジョシュ・シャピロ 州 司法 長官 が 率い た 2 年間 に 渡る 調査 の 結果 で あり 、 州 内 の 8 つ の 教区 の うち 、 6 つ の 教区 で 、 
    教会 職員 たち が 行っ た 「 組織 的 な 」 隠蔽 を 詳細 に 記し て いる 。 残り の 2 つ の 教区 は 、 別 の 大 陪審 報告 書 の 対象 と なっ た 。 

    「 ペンシルバニア 州民 は 、 ようやく これら の 教区 における 性的 虐待 の 規模 について 知る こと が でき ます 」 と 、 シャピロ 司法 長官 は 火曜日 の 記者 会見 で 語っ た 。 
    「 教会 指導 者 たち による 組織 的 な 隠蔽 を 、 私 たち は 初めて 理解 し 始める こと が できる の です 。 虐待 は あらゆる 教区 に 傷痕 を 残し まし た 。 手 の 込ん だ 隠蔽 が 行わ 
    れ て い た の です 。 教会 は どんな 犠牲 を 払っ て も 組織 を 守り まし た 」 

    大 陪審 員 たち は 報告 書 の 中 で 、 1 , 000 人 の 子ども たち ー その ほとんど が 少年 たち ー に対する 性的 虐待 の 疑い を 含む 、 50 万 ページ に およぶ 教会 の 内部 文書 
    を 召喚 、 検討 し 、 実際 の 犠牲 者 数 は 数 千 人 に 上る 可能 性 が ある こと を 示し た 。 

    「 アルコール や ポルノ によって 操ら れ た 子 も い た 。 加害 者 に 手淫 を さ せ られ たり 、 痴漢 行為 を さ れ た 子 も い た 」 と 、 報告 書 に 記さ れ て いる 。 

    「 口腔 性交 、 膣 性交 、 肛門 性交 を 強要 さ れ て い た 子 も い た 。 しかし 、 州 内 の あらゆる 所 で 、 何 に も まして 虐待 者 や 教会 組織 を 守る こと を 選ん だ 教会 指導 者 たち 
    によって 、 虐待 さ れ た 子供 たち は 全員 黙殺 さ れ た 」 

    教会 指導 者 は 子ども たち を 助ける こと より 、 スキャンダル を 回避 する こと を 選択 し た こと が 分かっ た 、 と 大 陪審 は 述べ た 。 

    「 聖職 者 たち は 小さな 男の子 や 女の子 を レイプ し て い た 。 彼ら の 責任 を 負う べき 聖職 者 たち は 、 何 も せ ず 、 全て を 隠蔽 し た 」 と 、 報告 書 は 記し て いる 。 

    報告 書 に よる と 、 隠蔽 の 結果 、 判明 し た 虐待 事件 の 多く が 時効 を 迎え て い た 。 しかし 、 大 陪審 に よる と 、 2 つ の 教区 で 2 人 の 聖職 者 が 、 出 訴期限 内 の 犯罪 で 告 
    発さ れ た 。 

    その うち 1 人 の 聖職 者 は 、 7 歳児 の 口腔 内 で 射精 し た 罪 で 告発 さ れ て いる 。 もう 1 人 の 聖職 者 に は 、 つい 最近 、 2010 年 まで 2 人 の 少年 を 数 年 にわたり 毎月 
    の よう に 暴行 し た 容疑 が かけ られ て いる 。 

    「 しかし 、 私 たち は ほんの 数 件 を 起訴 に 持ち込め た だけ で 満足 は し て い ない 。 それ は 私 たち が 発見 し た 児童 虐待 者 の ほんの 一 握り に すぎ ない の だ 」 と 、 報告 書 
    は 記し て いる 。 

    「 罰せ られる こと も 補償 さ れる こと も ない 、 全て の 犯罪 に対して 、 私 たち は 憤慨 し て いる 。 この 報告 書 は 、 私 たち にとって 唯一 の 頼み の 綱 な の だ 」 

    大 陪審 の 報告 書 に よる と 、 教会 の 内部 文書 で は 、 性的 暴行 は 「 不適切 な 行為 」 または 「 境界 問題 」 と 呼ば れ て い た 。 苦情 が あっ た 場合 に は 、 仲間 の 聖職 者 が 同 
    僚 から 聞き 取り を 行う 任務 を 負っ た 。 

    ある 聖職 者 を 解任 し なけれ ば なら ない 場合 、 教 区民 に その 理由 が 明かさ れる こと は 決して なかっ た 。 その 代り 、 その 聖職 者 は 「 病気 休暇 中 」 または 「 神経 衰弱 
    」 を 患っ て いる 、 と 伝え られ た 。 

    報告 書 に よる と 、 聖職 者 の 行い が 公 に なっ た 場合 、 そうした 行い は 警察 に 通報 さ れ なかっ た 。 その 代り 、 「 誰 も 彼 が 児童 虐待 者 だ と 知る 者 が い ない 」 他所 の 教 
    区 へ 異動 に なっ た 。 

    かつて ピッツバーグ で 長年 に 渡り 司教 を 務め た ドナルド ・ ワール 枢機卿 は 、 その 報告 書 の 中 で 、 聖職 者 の 異動 や 虐待 容疑 に対する 自ら の 処遇 を 非難 さ れ て おり 
    、 ワシントン 大司教 区 を 通じ た 声明 を 発表 し 、 司法 長官 は 自分 の 行動 を 誤っ て 特徴 づけ て いる と 非難 し た 。 

    「 事実 上 、 大小 さまざま な 方法 で 、 司法 長官 の 事務所 は この 報告 書 を 正しく 仕上げる こと より 、 公 に する こと に 関心 が あっ た 。 その よう な 焦点 は 、 保護 と 癒 や 
    し という 共通 の 目標 を 損なう もの だ 」 と 、 枢機卿 は 語っ た 。 

    「 ワール 枢機卿 が 明らか に し た よう に 、 この 報告 書 の 結果 は 忠実 な 信者 、 一般 市民 、 しかし 特に 虐待 を 生きのび た 人々 と その 家族 、 愛する 人々 にとって 試練 に 
    満ち た 時 を 生み出す でしょ う 。 被害 者 たち の 治癒 を 助ける こと 、 また その 一方 で 最も 弱い 立場 に いる 人々 が 守ら れる こと に 焦点 が 置か れる べき な の です 」 

    この 報告 書 で は 、 「 癌 検査 」 を 行う として 12 人 の 少年 たち の 体 を 触っ た こと を 告白 し た エリー 教区 の 聖職 者 を 含め た 犯罪 容疑 者 や 、 隠蔽 を 手助け し た 者 の 実 
    名 を 列挙 し て い た 。 罪 の 告白 後 、 その 聖職 者 は 15 年間 も 職務 を 続け た 。 

    アレン タウン 教区 の 別 の 聖職 者 は 、 少年 を 性的 に 虐待 し た こと を 告白 し た 。 しかし どちら の 教区 も 、 被害 者 にとって 「 その 体験 は 必ずしも 恐ろしい トラウマ に 
    は なら ない 」 と 結論 付け 、 被害 者 家族 に は 「 問題 を 公開 し て 自由 に 討論 する 機会 」 を 与える べき 、 と し た 。 

    グリーンズバーグ 教区 で は 、 聖職 者 が 17 歳 の 少女 を 妊娠 さ せ 、 結婚 証明 書 に 主 牧師 の 署名 を 偽造 し 、 その後 被害 者 を 離婚 し た 。 

    「 未成年 者 と 性 交渉 を 持っ た に も かかわら ず 、 子ども を もうけ た に も かかわら ず 、 結婚 し て 離婚 し た に も かかわら ず 、 その 聖職 者 は 職務 を 続ける こと を 許可 さ 
    れ た 」 と 、 報告 書 に 記さ れ て いる 。 

    ある 少年 は 、 教区 司祭 館 に ある ベッド の 上 に 立たさ れ 、 全裸 に さ れ 、 聖職 者 たち の ため に 十字架 上 の キリスト の ポーズ を とら さ れ た 、 と 火曜日 に 行わ れ た 記者 
    会見 で シャピロ 司法 長官 は 語っ た 。 聖職 者 たち は その後 被害 者 の 写真 を 撮影 し 、 

    「 その 写真 を 、 自分 たち が 教会 の 敷地 内 で 制作 し 共有 し て い た 児童 ポルノ の コレクション に 追加 し た の です 」 

    報告 書 に よる と 、 大 陪審 の 審議 中 に 、 証言 を 行っ た 被害 者 の 一 人 が 自殺 を 試み た 。 

    「 病院 の ベッド から 、 彼女 は たった 一つ 、 私 たち が 仕事 を やり遂げ 、 実際 に 何 が 起こっ た の か を 世界 に 知ら しめる こと を 望ん だ 」 と 、 報告 書 に 記さ れ て いる 。 

    「 私 たち は この 女性 と 、 自分 たち の 身 に 起き た こと を 語る こと で 自分 たち を さらけ出し た 、 その他 大勢 の 被害 者 たち に対して 恩義 を 感じ て いる 。 この 報告 書 が 
    、 私 たち の 借り を 返し て くれる こと を 願っ て いる 」 

    報告 書 の 全文 は こちら から 。 

    この 記事 は 英語 から 翻訳 さ れ まし た 。
