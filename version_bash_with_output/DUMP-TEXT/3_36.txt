    強姦 する こと を 許可 し て ほしい 」 と 伝え た 。 
    もちろん 悪い の は オマル で あり 、 オマル の 妹 は 何 も 悪く ない が モハメド は 犯人 に 同じ 苦痛 を 与え たかっ た の だ 。 

    そんな 無茶 な モハメド の 願い を 村長 は あっさり と 許可 。 村 の 評議 会 により 決定 さ れ 、 モハメド は ただちに オマル の 17 歳 の 妹 を 訪ね て いき 両親 の 目 の 前 で 強姦 
    し た 。 

    被害 を 受け た 女性 の 両方 の 母親 が 地元 警察 に 通報 し 、 この 事件 が 知れる と 警察 が 捜査 に 乗り出し 、 事件 と 関連 し て いる 村 の 人々 28 人 を 逮捕 。 村 の 人達 は 「 村 
    の 伝統 だ 」 と 語っ て いる と いう 。
